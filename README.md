# Projet-VidP : Pipeline Hybride de Stabilisation Vidéo
Projet VidP: Pipeline hybride de traitement vidéo.

Descrition : Le pipeline de stabilisation vidéo a pour but de transformer une séquence vidéo tremblante ou saccadée en une séquence fluide et stable, comme si elle avait été filmée avec un trépied ou un stabilisateur professionnel.
